# Contributing

Thank you for contributing to Oklix.

## Development

```bash
python -m venv .venv

pip install -r requirements.txt

pytest
```

## Coding Style

- Use Python 3.11+
- Follow PEP8
- Write unit tests
- Keep functions small
- Prefer composition over inheritance

## Pull Requests

- Create feature branch
- Ensure tests pass
- Update README if needed
